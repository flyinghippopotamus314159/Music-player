Each note is seperated by a ;
Each value for the note is seperated by a ,
These values are note (0-11 from A-G#), octave(1-4 corresponding to A2 onwards through to A6 onwards), length in beats
Before these notes is the bpm, a "," and "ignore"
E.g:
120,"ignore";0,2,1;2,2,0.5;4,2,0.5;5,2,0.5;7,2,0.5;9,2,0.5;11,2,0.5;0,3,2 Plays an Amaj scale
